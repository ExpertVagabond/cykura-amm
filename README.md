# Cykura AMM

Concentrated liquidity AMM protocol on Solana — fork and revival of the Cykura protocol, submitted for the Solana Graveyard Hackathon.

![Solana](https://img.shields.io/badge/Solana-9945FF?logo=solana&logoColor=white)
![License](https://img.shields.io/badge/License-MIT-green)

## Overview

Cykura AMM implements a Uniswap v3-style concentrated liquidity model on Solana. Liquidity providers allocate capital within custom price ranges, improving capital efficiency over traditional constant-product AMM designs.

## Hackathon

Submitted to the **Solana Graveyard Hackathon** as a revival of the dormant Cykura concentrated liquidity protocol.
